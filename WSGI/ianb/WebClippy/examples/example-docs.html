Welcome to Clippy!

WebClippy is a system to provide help on web pages. It's this floating bar right here. With it, you can provide instructions about what a person should do on the web, in that screen right down there.

To go to the next slide, hit that little button (or hit the link right there).

What Clippy Does

As you see, you've advanced to the next slide. You can still go back. Or continue on to the next slide.

What Clippy Doesn't Do

You'll notice Clippy doesn't do is advance automatically. Or figure out what you are doing. Or keep you from going back. It's pretty simple -- just a set of slides.

They are HTML slides, so of course you can still include any HTML in them that you want; bold, italic, and of course everything else (tables, etc). You can link to a page with <a href="..." target="docFrame">...</a>.

What Clippy Is Like Inside

Clippy slides are separated with horizontal rules, aka <hr>. The magic backward and forward links just link to the anchors #next and #prev. Or link to a page with #(number) like this.

Invoking Clippy

If you look at the URL up there, it points to frame.html. This is an empty frame, which Clippy fills in using the parameters doc (the starting document) and help (this help document).
