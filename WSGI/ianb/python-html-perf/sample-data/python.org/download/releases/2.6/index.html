Python 2.6

These are the alpha releases of Python 2.6. They are not suitable for production use. We're releasing it in order to solicit feedback from the wider Python development community. The NEWS file conains a listing of everything that's new in Python 2.6a1.

PEP 361 includes the schedule and will be updated as the schedule evolves. At this point, any testing you can do would be greatly appreciated.

Please report bugs at http://bugs.python.org

See also the license.

Python 2.6a1 Released: February 29th 2008
