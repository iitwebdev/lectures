Python 1.5 binaries for Windows NT and Windows 95
=================================================

Other places to look:

	http://www.python.org/download/download_windows.html
			Python for Windows download page
	ftp://ftp-icf.llnl.gov/pub/python
			LLNL's Python directory;
			NumPy-11.EXE is the Windows installer

	http://www.python.org/windows/

	http://starship.skyport.net/crew/mhammond/
			Mark Hammond's Python area on starship
	http://starship.skyport.net/crew/mhammond/1.5/
			Mark Hammond's Python 1.5 area on starship

	../wpy/		WPY, an alternative portable GUI

	../pc/		Binaries for Windows 3.1(1), DOS, OS/2


py152.exe	Installer for Python 1.5.2 (final)
py152c1.exe	Installer for Python 1.5.2 (release candidate 1)
py152b2.exe	Installer for Python 1.5.2 (second beta)
py152b1.exe	First beta.

pyth151.exe	Self-extracting installer program for Python 1.5.1.
		Includes HTML docs and Tcl/Tk support.
		For Tcl/Tk support, you need Tcl/Tk too;
		see ftp://ftp.sunlabs.com/pub/tcl/.
		Version 8.0p2 or later preferred.

python15.exe	Self-extracting installer program for Python 1.5. (Ditto)

pil.zip		PIL 0.3a1/0.3a2, compiled for Python 1.5b2 by Mark
		Hadfield.  Works with Python 1.5 (final) too.
		Repackaged by Guido van Rossum.

old/		Older releases (note that xfreeze is no longer
		needed with Python 1.5; NT support is included in the 
		1.5 source for freeze).
